# GTA-V - Macro

## Using this Macro is on your own riks
Using this macro is at your own risk. If Rockstar detects it, you could get banned from GTA Online or even all Rockstar Games. Always use macros responsibly and understand the risks involved.

## How to use this Macro
- You need to be have been installed Auto Hotkey V1 before you can run this Macro. [AHK V1 Installer](https://www.autohotkey.com/download/ahk-install.exe)<br>
- [download](https://github.com/TheSuperGiant/GTA-V---Macro/releases/download/DirectLinkAHKFile/GTA.V.ahk) the GTA.ahk file.
- Now rename the downloaded file for example "updater.ahk" or "gtaHelper.ahk".
- If you does not have an icon file download an icon file from internet.
- click right mouse on the ahk file you have renamed and choose: "Compile Script (GUI)..."
- Behind "Custom Icon" click there on browse
- Now click on your icon that you already have downloaded or already had on your pc.
- Click on Convert.
- Click on your new created EXE file and the Macro is working. Next time when you need to start the Macro again you have only to click on that new created exe file.
